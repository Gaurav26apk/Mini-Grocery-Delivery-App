# Mini Grocery Delivery App

A modern Android application developed in Kotlin that simulates a grocery delivery experience with OTP login, product browsing, cart management, checkout, and order confirmation.

## Features

- OTP-based Login (Dummy OTP: 1234)
- Product Search
- Category Filtering
- Add to Cart
- Increase / Decrease Quantity
- Remove Items from Cart
- Total Bill Calculation
- Checkout with Delivery Address
- Payment Method Selection
- Order Success Screen with Order ID
- Responsive UI
- Dark Mode Support
- RecyclerView Animations

## Tech Stack

- Kotlin
- Android XML
- RecyclerView
- ViewModel
- LiveData
- MVVM Architecture

## Project Structure

- `model/` - Data classes
- `adapter/` - RecyclerView adapters
- `viewmodel/` - ViewModels
- `utils/` - Dummy product data
- Activities for Login, Home, Cart, Checkout, and Order Success

## Dummy Login Credentials

- Mobile Number: Any valid 10-digit number
- OTP: `1234`

## Screens Included

- Login Screen
- Home Screen
- Cart Screen
- Checkout Screen
- Order Success Screen

## How to Run

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Run the app on an emulator or Android device.
